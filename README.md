# Makeunion — Портал (Проект 01)

Портал на **Next.js (App Router) + TypeScript**. Единая точка входа над продуктами
Ромы: каталог материалов, квиз, сопровождение и курсы. Готовые квиз и лендинги
**не пересобираются** — подключены ссылками.

## Запуск

```bash
cd portal
npm install
npm run dev
```
Откройте http://localhost:3000

Сборка прода: `npm run build` затем `npm start`.

## Что внутри

| Маршрут | Что это |
|---|---|
| `/` | **Главная = каталог материалов** + входы по ролям (селлер/менеджер/ученик) |
| `/razbory/[slug]` | Страница отдельного материала |
| `/admin/materials` | Упрощённая админка: добавление материала (форма + API) |
| `/kursy` | Заготовка: ссылки на курсы Tilda |
| `/final` | Заготовка: финальная страница «Спасибо» |
| `/api/materials` | `GET` список, `POST` добавление материала |

## Структура

```
app/            маршруты и страницы (App Router)
components/     UI-компоненты (шапка, карточки, формы, фильтр)
lib/
  links.ts      все внешние ссылки в одном месте
  types.ts      модель данных Material/Category (под будущий Prisma)
  materials.ts  доступ к данным каталога (сейчас JSON, легко заменить на БД)
data/
  materials.json  демо-каталог
content/
  PORTAL.md               ТЗ и карта структуры
  DOBAVLENIE-MATERIALOV.md инструкция: как добавлять материалы/продукты/формы
```

## Ключевые решения
- **Дизайн:** Dela Gothic One + Golos Text, цвета `#ff006b/#591ba1/#fa003a`,
  тёмная тема. Токены — в `app/globals.css`.
- **Данные:** демо-JSON с абстракцией в `lib/materials.ts`. Переход на Prisma+БД
  не затрагивает страницы. Запись в JSON работает локально; на Vercel ФС read-only
  → там источником станет БД.
- **Готовое — ссылками:** квиз и лендинги открываются на своих доменах
  (`lib/links.ts`).

Подробности — в `content/PORTAL.md`.

# Дизайн-система портала

**Источник правды:** боевой сайт **[romarayt.ru/ozon](https://romarayt.ru/ozon)** («Авторская система продаж на Ozon»).
Токены сняты 1-в-1 с настоящего CSS сайта (Tilda ZeroBlock). Мусор библиотек
(Bootstrap/jQuery UI/select2/glyphicons) и декоративные фоны отдельных промо-блоков
Тильды — отброшены (см. раздел «Что отброшено»).

Система совпадает с Ozon-квизом: тёмный фон, **Golos Text**, градиент **#591BA1 → #FA003A**,
розовый **#ff006b**. Токены живут в `app/globals.css` (префикс `--rr-*`).

---

## Шрифт

**Golos Text** — и заголовки, и текст. Веса **400 / 500 / 600 / 700**
(на сайте — @font-face с tildacdn: Regular/Medium/SemiBold/Bold).
Подключён через `next/font/google` (`Golos_Text`, subsets latin+cyrillic) в `app/layout.tsx`
как `--font-golos`. Локальные файлы не нужны.

## Радиусы · Отступы · Контейнер

- **Радиусы:** `--rr-radius-sm 8` · `--rr-radius-md 16` · `--rr-radius-lg 20` · `--rr-radius-xl 30` · `--rr-radius-pill 100` (доминанты сайта — 16/8/10/20/30).
- **Отступы:** шкала `4 / 8 / 12 / 16 / 20 / 24 / 32 / 48 / 64 / 80`.
- **Контейнер:** `--container 1200px` (`--container-narrow 880px`); брейкпоинты Tilda 960 / 640 / 480.

## Компоненты

- **Кнопка primary:** заливка `--rr-grad-primary-h` (#591BA1→#FA003A), текст белый, Golos 600, радиус `lg` (20), паддинг `16×32`, свечение `--rr-shadow-glow-purple`. Hover — `brightness(1.08)` + подъём 1px.
- **Кнопка secondary:** прозрачная, `1px solid #ffffff2e`; hover — граница `--rr-pink`.
- **Бейдж:** фон `#ff006b1f`, граница `#ff006b4d`, текст `--rr-pink`, uppercase, радиус pill.
- **Карточка:** фон `--rr-bg-card`, граница `--rr-border-muted`, радиус `lg`.
- **Поля ввода:** фон `--rr-bg-deep`, focus — граница `--rr-pink` + кольцо `rgba(255,0,107,.18)`.
- **Фон страницы:** радиальное фиол. свечение сверху (`#591ba159`) + малиновое снизу (`#fa003a33`) + база `#030510`.
